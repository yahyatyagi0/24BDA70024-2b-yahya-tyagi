## Aim
To create a dynamic product filter that sorts items based on dropdown selection and allows users to add products to a cart using Next.js.

---

## Objectives
- Implement filterable and sortable product data  
- Create a dropdown UI for sorting (price and name)  
- Design responsive product cards  
- Add an “Add to Cart” feature with cart count  
- Apply smooth UI transitions  

---

## Technologies Used
- **Next.js (App Router)**
- **React Hooks (useState, useMemo)**
- **JavaScript (ES6)**
- **CSS Grid & Flexbox**
- **HTML5**

---

## About the Experiment
This experiment demonstrates the use of **Next.js with React hooks** to build a dynamic and interactive user interface.  
Product data is stored in arrays and manipulated using JavaScript methods such as `sort()` and `map()`.  
The application includes a responsive layout using CSS Grid and a basic cart system managed through React state.

---

## Features
- Sort products by **price (Low–High / High–Low)**  
- Sort products by **name (A–Z)**  
- Responsive product grid  
- Add products to cart  
- Cart count updates in real time  
- Clean and modern UI
